# data-co-pliot-vue3

> 需要安装

- npm 
- node
- git
- vue3
- element-plus
- vite

> 登陆页面
- 需要后端api端点/login
- 没有请求头 
- 发送一个POST请求到后端的API端点，携带用户名和密码作为请求体。
- 登录成功，后端应该返回一个token，前端会将其存储在localStorage中


## 安装

```bash
npm install
```

### 运行

```bash
npm run dev
```

#### 构建

```bash
npm run build
```

## 使用

```bash
git 
cd 
npm i
npm run dev
```
